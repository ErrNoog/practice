# Мои учебные проекты на C и ассемблере NASM

Привет! Я начинающий разработчик, учусь системному программированию под Linux.

Это мои первые самостоятельные проекты. Они проходили только базовые тесты и не претендуют на профессиональные программы. 

Файл stud_io.inc для программ на ассемблере был взят из: http://www.stolyarov.info/books/extra/stud_io_inc. Он сокращает системные вызовы для: печати символа, ввода символа, символа перевода строки, выхода из программы.

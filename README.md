Все программы написаны исключительно с целью получения практического опыта. Они не проходили особых тестов, поэтому могут работать некорректно!  

Файл stud_io.inc для программ на ассемблере был взят из: http://www.stolyarov.info/books/extra/stud_io_inc. Он сокращает системные вызовы для: печати символа, ввода символа, символа перевода строки, выхода из программы.
